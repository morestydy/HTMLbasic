# 选择器

帮助精准选中想要的元素

## 简单的选择器

1. ID 选择器
2. 元素选择器
3. 类选择器
4. 通配符选择器

*,选中所有元素

5. 属性选择器

根据属性名和属性值选中元素

6. 伪类选择器

选中某些元素的某种状态

1) hover:鼠标悬停时状态

2) active: 激活状态,鼠标按下状态

3) link: 超链接未访问时的状态

4) visited:超链接访问后的状态

书写顺序
